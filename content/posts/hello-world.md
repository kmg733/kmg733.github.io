## 환영합니다!

이것은 Next.js + TypeScript + Markdown으로 만든 블로그의 첫 번째 포스트입니다.

### 주요 기능

- **Markdown 지원**: `.md` 또는 `.mdx` 파일로 글 작성
- **정적 사이트 생성**: GitHub Pages에 배포 가능
- **TypeScript**: 타입 안정성 보장
- **Tailwind CSS**: 빠른 스타일링

### 코드 예시

